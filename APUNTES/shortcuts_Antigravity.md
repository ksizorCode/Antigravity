# Atajos de teclado de VSCode y Antigravity
Al ser un Antigravity y fork de Microsoft Visual Studio Code gran parte de los atajos de teclado son herdados de este sistema, aunque obviamente las funcionalidades extra que la aplicación aporta cuentan con sus propios shortcuts:



## Atajos generales del sistema

Ctrl + A    Seleccionar Todo
Ctrl + C    Copiar
Ctrl + Z    Deshacer

Win + V     Historial de Portapapeles


## Atajos Comunes VSCode & Antigravity
Ctrl + L   Selecciona Linea
Alt + Shift + Abajo        Selecciona Liguiente linea
Alt + Shift + Abajo        Selecciona Liguiente linea
Alt + Shift + Arriba       Selecciona Anterior linea
Alt + Scroll               Permte escribir en varias lineas de forma simultánea

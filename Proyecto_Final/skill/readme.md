# Skill para desarrollar el proyecto

El siguiente documento `skill.md` recoge las pautas del ejercicio para llevarlo a cabo de una forma más o menos automatizada.

Como ejecutarlo con *Google Antigravity*. Pasos a seguir:

1. Antes de empezar describe claramente el proyecto que vas a llevar a cabo.

2. Guarda este skill en Google Antigravity.

3. Ejecutalo dandole el contecto anterior (paso 1).
